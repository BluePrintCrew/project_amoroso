# Amoroso

> 가구 전자상거래 플랫폼

🌐 **배포 주소**: [amoroso.blue](https://amoroso.blue)  
📅 **개발 기간**: 2025.01 ~ 2025.06 (6개월)

## 주요 기능

- 🔐 소셜 로그인 (구글, 카카오, 네이버)
- 🛋️ 가구 상품 관리 및 검색
- 🛒 장바구니 & 주문 관리
- 💳 PortOne 결제 연동
- ⭐ 리뷰 시스템
